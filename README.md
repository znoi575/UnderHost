## UnderHost是什么来头？

先简单介绍一下背景。UnderHost成立于2007年，总部在加拿大，但业务遍布全球12个数据中心。这不是那种只会喊口号的小作坊——他们在香港、荷兰、俄罗斯、加勒比群岛、美国等地都有自己的服务器节点。

这家公司最大的卖点就是"离岸主机"（Offshore Hosting）。什么意思呢？简单说就是把服务器放在那些对版权、内容审查相对宽松的法律管辖区。这样一来，即使有人发DMCA投诉（数字千年版权法投诉），只要你的内容不违反当地法律，服务商也不会轻易把你的站点下线。

当然，这不代表可以为所欲为。UnderHost明确禁止垃圾邮件、恶意软件、钓鱼网站、儿童色情等内容。但对于合法的灰色地带内容——比如成人内容、某些类型的软件分享、或者政治敏感话题——他们的态度要开放得多。
